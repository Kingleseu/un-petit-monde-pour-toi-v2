## 🚀 Comment Utiliser

### Démarrer le projet
```bash
npm install
npm run dev
```

### Personnaliser les couleurs
Éditez `stepConfigs` dans `Background.tsx`

### Modifier les animations
Cherchez `transition={{ duration: X }}` dans les composants

### Ajouter des messages de transition
Éditez le tableau `messages` dans `TransitionScreen.tsx`
